# 📄 Log Automation Script

## 📌 Description
Automates copying and analysis of Apache, Spark, and Hadoop logs.

## 🚀 Features
- Copies logs from source to destination
- Extracts ERROR/WARN/FATAL messages
- Supports text and JSON logs
- Formats logs into a unified structure

## 🦾 Usage
```
python log_automation.py
```
